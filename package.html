contains the classes for describing trains. The Train class contains the information about a train - its name, its identity, the lead engine number, the crew assigned to it, its status. etc. Eventually, the plan is to tie this in with the car routing program so that the dispatcher can see the length, its wheel report and other things. One Train class is instantiated for each train. The roster holds the list of all trains. There is a sub-list for each state that a train can be in.

Package Specification

This package is tied into cats.layout.items because Trains need to know about Blocks for implementing the train movement.
